# Favorite Icon

The Favorite Icon represents a favorite or starred item in the interface. It's commonly used to indicate items that have been marked as favorites or to provide a way to favorite/unfavorite items.

## Usage

```tsx
import { FavoriteIcon } from '@mongodb-js/compass-components';

function MyComponent() {
  return <FavoriteIcon />;
}
```

### Default

<Story name="Default">{() => <FavoriteIcon />}</Story>

## Guidelines

- The icon is designed to be used at its original size (72x72 pixels)
- It automatically adapts to light/dark mode
- It's SVG-based for crisp rendering at any size
- It should be used to represent favorite/starred items or actions
- It should always be accompanied by text or a tooltip for screen readers
