# Document Icon

The Document Icon represents a document or file in the interface. It's commonly used to indicate document-related actions or to represent document items in lists.

## Usage

```tsx
import { DocumentIcon } from '@mongodb-js/compass-components';

function MyComponent() {
  return <DocumentIcon />;
}
```

## Guidelines

- The icon is designed to be used at its original size (72x72 pixels)
- It automatically adapts to light/dark mode
- It's SVG-based for crisp rendering at any size
- It should be used to represent documents, files, or document-related actions
- It should always be accompanied by text or a tooltip for screen readers
