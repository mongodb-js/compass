# Placeholder

The Placeholder component is a loading state visualization that displays an animated gradient.
It's commonly used to indicate loading states for text content, maintaining the layout structure
while data is being fetched or processed.

## Key features

- Animated gradient effect for visual feedback
- Configurable width and height
- Random width generation based on character ranges
- Customizable gradient colors

## Usage

The Placeholder component is perfect for:

- Loading states in forms
- Content placeholders while data is being fetched
- Maintaining layout stability during async operations

```tsx
import { Placeholder } from './placeholder';

// Basic usage
<Placeholder />

// With custom dimensions
<Placeholder width="200px" height="20px" />

// With character range
<Placeholder minChar={5} maxChar={15} />
```
