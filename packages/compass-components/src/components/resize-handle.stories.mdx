# ResizeHandle

The ResizeHandle component is used to create resizable panels and sidebars, typically in conjunction with the `re-resizable` library.

## Usage

```tsx
import { ResizeHandle, ResizeDirection } from '@mongodb-js/compass-components';
import { Resizable } from 're-resizable';

<Resizable
  defaultSize={{ width: '300px', height: '100%' }}
  minWidth="200px"
  maxWidth="500px"
  enable={{ right: true }}
  handleComponent={{
    right: (
      <ResizeHandle
        direction={ResizeDirection.RIGHT}
        value={300}
        minValue={200}
        maxValue={500}
        onChange={() => {}}
        title="sidebar"
      />
    ),
  }}
  handleStyles={{ right: { right: '-9px' } }}
>
  <div>Sidebar content</div>
</Resizable>;
```
