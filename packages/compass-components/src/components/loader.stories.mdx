# Loader

The Loader components provide visual feedback for loading states.

## Usage

```tsx
import { SpinLoader, SpinLoaderWithLabel, CancelLoader } from '@mongodb-js/compass-components';

<SpinLoader size={12} />
<SpinLoaderWithLabel progressText="Loading..." />
<CancelLoader progressText="Loading..." cancelText="Cancel" onCancel={() => {}} />
```
