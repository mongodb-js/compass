# Document Actions Group

The Document Actions Group component provides a set of action buttons for document operations. It's commonly used in document lists and views to provide quick access to common document operations like edit, copy, clone, and remove.

## Usage

```tsx
import { DocumentActionsGroup } from '@mongodb-js/compass-components';

function MyComponent() {
  return (
    <DocumentActionsGroup
      onEdit={() => console.log('Edit clicked')}
      onCopy={() => console.log('Copy clicked')}
      onClone={() => console.log('Clone clicked')}
      onRemove={() => console.log('Remove clicked')}
    />
  );
}
```

## Guidelines

- The component is designed to be used within a document list or view
- Action buttons are hidden by default and appear on hover
- Each action button has a tooltip explaining its function
- The component supports expand/collapse functionality for nested documents
- Insights can be displayed to provide additional context about the document
- All buttons are accessible and have appropriate ARIA labels
