# KeylineCard

The KeylineCard component is a simple card component with a keyline border.
It supports different content styles and can be used to display various types of content.

## Usage

```tsx
import { KeylineCard } from '@mongodb-js/compass-components';

function KeylineCardExample() {
  return (
    <KeylineCard contentStyle="clickable">
      <h3>Card Title</h3>
      <p>This is a paragraph of text in the card.</p>
    </KeylineCard>
  );
}
```
