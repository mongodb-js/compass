# ResizableSidebar

The ResizableSidebar component provides a resizable sidebar panel that can be
dragged to adjust its width. It's commonly used for navigation menus, settings panels,
or any content that needs to be collapsible and resizable.

## Key features

- Draggable resize handle
- Configurable initial and minimum widths
- Smooth resize animations
- Optional new theme styling
- Maintains layout structure during resize

## Usage

The ResizableSidebar component is perfect for:

- Navigation menus
- Settings panels
- Sidebar content that needs to be resizable
- Collapsible panels

```tsx
import ResizableSidebar from './resizeable-sidebar';

// Basic usage
<ResizableSidebar>
  <YourSidebarContent />
</ResizableSidebar>

// With custom width
<ResizableSidebar initialWidth={400} minWidth={250}>
  <YourSidebarContent />
</ResizableSidebar>

// With new theme
<ResizableSidebar useNewTheme={true}>
  <YourSidebarContent />
</ResizableSidebar>
```

## Best Practices

1. **Minimum Width**: Always set a minimum width that ensures your content remains readable
2. **Initial Width**: Set an initial width that works well for most users
3. **Content Overflow**: Ensure your content handles overflow gracefully
4. **Responsive Design**: Consider how the sidebar behaves on different screen sizes
