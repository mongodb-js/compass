# mongodb-ace-theme-query [![][npm_img]][npm_url]

MongoDB ACE Theme for Query Bar

## Installation

```
npm install --save mongodb-ace-theme-query
```

## Usage

```javascript
import 'mongodb-ace-theme-query';
editor.setTheme("ace/theme/mongodb-query");
```
